## 現在のタスク
- セグメント番号: {segment_index}
- 時間範囲: {time_range}

## 出力要件
高品質な動画生成プロンプトを作成してください。必ず以下を含めます:

1. 映像内容:
   - 具体的な人物/対象: 外見、姿勢、表情
   - 環境の詳細: 背景、光、天候/雰囲気
   - 動作: 方向、速度、リズム

2. カメラ言語:
   - ショット種別: 寄り/中景/引き/追従/パン
   - カメラ移動: 押し/引き/横移動/回転
   - 構図: 主体の位置、前景/背景

3. ビジュアルスタイル:
   - 色: 暖色/寒色/高コントラスト
   - ライティング: 順光/逆光/側光
   - 質感: 写実/アニメ/サイバーパンク

4. 連続性（重要）:
   - 最初のセグメントなら、開始フレームを明確にする。
   - 連続性サマリや参照フレームがある場合、必ずそこから継続する。
   - 終了フレームを明確にして、次のセグメントに備える。

5. 音楽の連続性（重要）:
   - セグメント間で一貫した音楽アイデンティティ（同じジャンル/モチーフ/BPM/核となる楽器編成）を保持する。
   - 変えてよいのは強弱・レイヤー・アレンジのみ。急な変化は避ける。
   - 次セグメントへの音楽遷移を明確に（滑らかで動機づけのある遷移）。
   - video_prompt と continuity_notes の中に "MUSIC:" の説明を必ず入れる。

## 出力形式（JSON）
```json
{{
  "script": "セグメントの詳細脚本（人間向け）",
  "video_prompt": "動画生成プロンプト（動画モデル向け）。主体 + 環境 + 動作 + カメラ + スタイルを含め、200-300語程度。",
  "start_frame_description": "連続性のための開始フレーム説明",
  "end_frame_description": "次のセグメントが継続すべき終了フレーム説明",
  "continuity_notes": "連続性メモ: 役者の状態、位置、道具、音楽など"
}}
```

注意: video_prompt は抽象ではなく、具体的で視覚的に。
